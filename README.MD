# Air Touch Controller

## 01 Introduction

A simple prototype for hand gesture detection, wherein the up-arrow key is pressed upon detecting the index finger, is the cornerstone for groundbreaking technology triggered by hand gestures. This innovation transcends traditional controls, offering a seamless interface for interacting with various devices. Imagine effortlessly turning on the air-conditioner, adjusting the TV volume, or dimming the lights‚Äîall with intuitive hand movements. This prototype marks the inception of a transformative era where human-computer interaction is redefined, promising unparalleled convenience and efficiency in our daily lives.

## 02 Technology Stack

- Python 3.6
- OpenCV
- pynput

## 03 Setup

- Install OpenCV

  ```
  pip install opencv-python
  ```

- Install pynput

  ```
  pip install pynput
  ```

## 04 Usage

```
python source/main.py
```
